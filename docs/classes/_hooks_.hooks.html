datasets

  • datasets(types: ChartType | ("line" | "bar" | "horizontalBar" | "radar" | "doughnut" | "polarArea" | "bubble" | "pie" | "scatter" | DatasetHook)[], general?: string): this
    • Defined in src/hooks.ts:133

    Set the dataset options. If a single string is provided, all the datasets will will be specified to the desired type. If an array is provided, if the inner element is a string, it will be used as a type, otherwise, the options will be applied, in both cases, to the nth position. Keep in mind that if the array length is less than the datasets, it will start from the start again.
